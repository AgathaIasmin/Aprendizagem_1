Banco de Dados

O que é um banco de dados?

Um banco de dados é uma coleção organizada de dados que permite armazenar, gerenciar e recuperar informações de forma eficiente.

Banco de dados relacional

Um banco de dados relacional armazena dados em tabelas inter-relacionadas, utilizando um modelo baseado em relações. Exemplos incluem MySQL e Oracle.

Banco de dados não relacional

Um banco de dados não relacional não utiliza tabelas rígidas e é ideal para grandes volumes de dados. Exemplos incluem MongoDB e Cassandra.

Diagrama Conceitual DER

Um diagrama de entidade-relacionamento que representa entidades e seus relacionamentos graficamente.

Diagrama Lógico MER

Um modelo entidade-relacionamento mais detalhado, que representa entidades, atributos e relacionamentos de forma lógica.

Exemplos de Diagramas

  
    Diagrama DER:  
    [Cliente] -- (Faz) --> [Pedido]  
    [Pedido] -- (Contém) --> [Produto]  

    Diagrama MER:  
    [Cliente]  
      - ID_cliente (PK)  
      - Nome  
      - Endereço  

    [Pedido]  
      - ID_pedido (PK)  
      - Data  
      - ID_cliente (FK)  

    [Produto]  
      - ID_produto (PK)  
      - Nome  
      - Preço  

    Relacionamentos:  
    - Cliente (1) ---- (N) Pedido  
    - Pedido (N) ---- (N) Produto

Bancos de dados utilizados pela empresa

Exemplos comuns incluem: MySQL, PostgreSQL, MongoDB, Microsoft SQL Server, Oracle.

O que é um dicionário de dados?

Um dicionário de dados armazena informações sobre os dados utilizados em um sistema, incluindo descrições de cada item de dados.

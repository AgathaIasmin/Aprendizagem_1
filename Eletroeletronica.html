Arduino Uno

O Arduino Uno é uma placa de microcontrolador baseada no ATmega328P. Ela possui 14 pinos digitais de entrada/saída, 6 entradas analógicas, uma conexão USB, um conector de alimentação, um cabeçalho ICSP e um botão de reset. É a placa mais comum utilizada para aprender e experimentar com eletrônica.

Especificações:

  • Microcontrolador: ATmega328P
  • Tensão de Operação: 5V
  • Tensão de Entrada (recomendada): 7-12V
  • Pinos Digitais de I/O: 14
  • Pinos de Entrada Analógica: 6
  • Memória Flash: 32 KB (ATmega328P)

Entrada analógica no arduíno

As entradas analógicas no Arduino permitem a leitura de sinais analógicos, que podem variar continuamente entre dois valores. O Arduino pode ler essas entradas através de seus pinos analógicos, que convertem o sinal analógico em um valor digital.

Como funciona:

O Arduino usa um ADC (Conversor Analogico-Digital) para converter o sinal analógico. Por exemplo, se você conectar um potenciômetro a um dos pinos analógicos, você pode ler o valor que varia de 0 a 1023, representando uma tensão de 0V a 5V (para o Arduino Uno).

Exemplo de Código:

  
            int sensorPin = A0; // Pino analógico onde o sensor está conectado  
            int valorSensor;  
            
            void setup() {  
                Serial.begin(9600); // Inicializa a comunicação serial  
            }  
            
            void loop() {  
                valorSensor = analogRead(sensorPin); // Lê o valor do pino analógico  
                Serial.println(valorSensor); // Envia o valor para o monitor serial  
                delay(500); // Espera meio segundo  
            }
